# CLAUDE.md

## Repo Notes

- This is a static Gomoku web game hosted on GitHub Pages.
- The project uses plain HTML, CSS, and JavaScript.

## Important Rules

- If you modify `style.css` or `script.js`, also update the asset version in `index.html`.
- Use cache-busting query parameters for frontend assets because GitHub Pages caching can delay CSS and JS updates.
- Be careful not to break the mobile layout.
- Keep the current game rules and simple robot behavior unless the user explicitly requests gameplay changes.
- Keep SEO-related files such as `robots.txt`, `sitemap.xml`, social preview assets, and metadata in good shape.
